# BankuishChallenge
Android App that lists Github repositories by kotlin

## Tech stack
- Kotlin
- Koin
- JetPack
  - LiveData
  - Lifecycle
  - ViewModel
  - Navigation Component
  - Data Binding
  - Paging 3
- Architecture
  - MVVM Architecture
- Picasso
- Retrofit2
